# atnplex/.github

Public organization profile and shared community health files for the `atnplex` GitHub organization.

## Purpose

This repository provides the public-facing defaults used across `atnplex`, including:

- organization profile content
- issue templates (bug reports, feature requests, support, questions)
- pull request templates
- community health defaults

## Internal automation

Private reusable workflows, CI/CD standards, runner strategy, and shared automation live in the private repository: [atnplex/.github-private](https://github.com/atnplex/.github-private)
